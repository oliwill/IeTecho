## 文档维护原则

- 产品边界变更时，同步更新 `README.md`、`CLAUDE.md`、PRD 和相关设计文档。
- 页面或路由变更时，同步更新 `architecture.md`、`runbook.md`、`handoff.md`。
- 新增云开发、环境变量或 AI API 时，同步更新 `runbook.md` 和 `CLAUDE.md`。
- 阶段性完成后更新 `changelog.md`。
